## Overview
Pilot Angels is a role-based scheduling application built using Mendix. 
It supports request intake, flight scheduling, pilot and aircraft management,
availability tracking, and confirmation workflows.

## Features
- Role-based access control
- Request intake and management
- Flight scheduling and confirmation
- Pilot and aircraft management
- Availability and flight logs
- View matches workflow

## Tech Stack
- Mendix Studio Pro
- Mendix Cloud
- UML (Class, Sequence, State Diagrams)
